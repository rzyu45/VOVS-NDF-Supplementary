# VOVS-NDF Supplementary Material

This repository contains the LaTeX source and compiled PDF for the supplementary material accompanying the manuscript "Acceleration of Gas-to-Power Cascading Fault Simulation Using Variable-Order-Variable-Step Numerical Differentiation Formulae".

## Files

- `main.pdf` - compiled supplementary material.
- `main.tex` - LaTeX entry point.
- `abbr.tex` - macro and abbreviation definitions used by the supplement.
- `supp.bib` - bibliography for the supplementary material.
- `fig/` - figure assets used by the supplement.

## Build

Run `latexmk` from the repository root to compile `main.tex` into `main.pdf`.
